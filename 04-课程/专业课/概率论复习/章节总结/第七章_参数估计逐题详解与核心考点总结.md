# 第七章 参数估计：核心考点总结与习题逐题详解

第七章可以说是整个数理统计中**套路最深、得分最稳**的一章。它主要解决一个问题：“我知道总体是什么分布，但我不知道里面的参数（比如 $\mu, \sigma^2, p, \theta$），怎么用样本去猜它们？”

本章只有两大核心板块：
1.  **怎么猜？**（两种估计方法：矩估计、最大似然估计）
2.  **猜得准不准？**（三种评价标准：无偏性、有效性、相合性）

---

## 🎯 第一部分：第七章核心考点极速总结

### 1. 矩估计法 (Method of Moments)
**核心思想**：“让样本的矩等于总体的矩”。
*   **第一步**：算出总体的期望 $E(X)$。这个期望里肯定包含未知参数。
*   **第二步**：强行令 $E(X) = \bar{X}$（样本均值）。
*   **第三步**：解方程，把未知参数用 $\bar{X}$ 表示出来。
*(如果有一个参数，只用 $E(X)=\bar{X}$；如果有两个参数，就要追加 $E(X^2) = A_2 = \frac{1}{n}\sum X_i^2$ 联立方程组。)*

### 2. 最大似然估计法 (Maximum Likelihood Estimation, MLE)
**核心思想**：“存在即合理，既然这组样本出现了，那就说明能让这组样本出现的概率（似然函数）是最大的”。
*   **第一步：写出似然函数 $L(\theta)$**。其实就是把所有样本的概率密度函数连乘起来：$L(\theta) = \prod_{i=1}^n f(x_i; \theta)$。
*   **第二步：取对数 $\ln L(\theta)$**。（为了把连乘变成连加，方便求导）。
*   **第三步：求导并令其等于 0**。即解方程 $\frac{d}{d\theta} \ln L(\theta) = 0$，解出的 $\theta$ 就是 MLE。
> [!WARNING]
> **最大似然的“特殊情况”（必考！）**
> 如果求导后发现导数永远大于 0 或永远小于 0（单调函数），说明极值在**边界**上！此时不用求导，直接看未知参数的取值范围。比如均匀分布 $U(0, \theta)$，$\theta$ 必须比所有样本都大，要让 $L(\theta) = \frac{1}{\theta^n}$ 最大，$\theta$ 要尽量小，所以直接取样本里的最大值 $\max(X_i)$。

### 3. 估计量的评价标准
*   **无偏性 (Unbiasedness)**：猜的平均结果得是准的。即 $E(\hat{\theta}) = \theta$。
    *(常考套路：算 $E(\hat{\theta})$ 的时候，狂用 $E(\bar{X})=\mu$, $E(S^2)=\sigma^2$)*
*   **有效性 (Efficiency)**：大家都是无偏的，谁的方差小谁就更有效。即比较 $D(\hat{\theta}_1)$ 和 $D(\hat{\theta}_2)$。

## 📝 第二部分：第 59-60 页习题逐题详细解析

下面为你逐一拆解图片上的 10 道题目。你的红笔过程写得非常精彩，大部分都是完全正确的，我来帮你规范化步骤和补全逻辑。

### 选择/填空题部分

**1. 考察 $\sigma^2$ 的无偏估计**
*   **解析**：如果 $\mu$ 已知，最好的估计是 $\frac{1}{n}\sum(X_i-\mu)^2$。如果 $\mu$ 未知（绝大多数情况），为了消除用 $\bar{X}$ 代替 $\mu$ 带来的偏差，必须除以 $n-1$，即 $S^2 = \frac{1}{n-1}\sum(X_i-\bar{X})^2$。
*   **答案选 (A)**：当 $\mu$ 已知时，$\frac{1}{n}\sum(X_i-\mu)^2$ 是无偏的。

**2. 考察常见统计量的无偏性**
*   **解析**：$S^2$ 是 $\sigma^2$ 的无偏估计（铁律）。但请注意，开根号后 $E(S) \neq \sigma$，所以 $S$ 不是无偏的。同时 $\bar{X}$ 和 $S$ 只在正态总体下独立。
*   **答案选 (C)**。

**3. 指数分布变形的矩估计**
*   **解析**：密度为 $e^{-(x-\theta)}$，通过积分求期望 $E(X) = \int_\theta^\infty x e^{-(x-\theta)} dx$（可用换元积分），算出来 $E(X) = \theta + 1$。
令 $E(X) = \bar{X}$，得到 $\theta + 1 = \bar{X}$，解得 $\hat{\theta} = \bar{X} - 1$。红笔完全正确。

**4. 二项分布 $B(k,p)$ 双参数的矩估计**
*   **解析**：这是这几题里计算量最大的。已知 $\mu = kp$, $\sigma^2 = kp(1-p)$。根据方差公式 $E(X^2) = D(X) + [E(X)]^2 = kp(1-p) + k^2p^2$。
建立方程组：
1) 一阶矩：$kp = A_1$ (即 $\bar{X}$)
2) 二阶矩：$kp(1-p) + k^2p^2 = A_2$
把 (1) 代入 (2) 得到：$A_1(1-p) + A_1^2 = A_2 \implies A_1(1-p) = A_2 - A_1^2 \implies 1-p = \frac{A_2 - A_1^2}{A_1}$。
所以 $\hat{p} = 1 - \frac{A_2 - A_1^2}{A_1}$。再代回 (1) 可解出 $\hat{k}$。红笔推导完美。

### 计算证明题部分

**5. 证明线性组合是无偏估计，并比较有效性**
*   **无偏性**：$E(W) = E(\sum a_i X_i) = \sum a_i E(X_i) = \mu \sum a_i$。因为 $\sum a_i = 1$，所以 $E(W) = \mu$，是无偏估计。$\bar{X}$ 显然也是无偏的。
*   **有效性比较**：$D(\bar{X}) = \frac{\sigma^2}{n}$。$D(W) = D(\sum a_i X_i) = \sum a_i^2 D(X_i) = \sigma^2 \sum a_i^2$。
我们要证 $\bar{X}$ 更有效，即证 $\frac{\sigma^2}{n} \le \sigma^2 \sum a_i^2$，也就是要证 $\sum a_i^2 \ge \frac{1}{n}$。
利用柯西不等式：$(\sum a_i \cdot 1)^2 \le (\sum a_i^2)(\sum 1^2) \implies 1^2 \le (\sum a_i^2) \cdot n$，得证！

**6. 证明 $T^2$ 和 $S^2$ 是无偏估计**
*   (1) $\mu$ 已知时：$E(T^2) = E[\frac{1}{n}\sum(X_i-\mu)^2] = \frac{1}{n}\sum E(X_i-\mu)^2 = \frac{1}{n}\sum D(X) = \frac{1}{n} \cdot n\sigma^2 = \sigma^2$。
*   (2) $\mu$ 未知时：$E(S^2) = \sigma^2$ 的证明在之前的第六章总结里讲过（利用 $\chi^2$ 分布自由度为 $n-1$ 来推导）。

**7. 均匀分布 $U(0, \theta)$ 的最大似然估计及评价**
*   **(1) 求极大似然**：$L(\theta) = \prod \frac{1}{\theta} = \frac{1}{\theta^n}$。注意前提条件是 $0 \le X_i \le \theta$ 对于所有 $i$ 成立，也就是 $\theta \ge \max(X_1...X_n)$。要让 $\frac{1}{\theta^n}$ 最大，$\theta$ 必须尽量小，所以直接取能取到的最小值，即 $\hat{\theta} = \max(X_1, X_2, \dots, X_n)$。
*   **(2) 证明有偏**：这就是我们刚做过的第六章 52 页第七题！我们算过极大值的期望：$E(\max) = \frac{n}{n+1}\theta \neq \theta$。所以它是有偏的。（虽然它有偏，但它是 MLE，说明 MLE 不一定是无偏的）。

**8. 分段函数的矩估计与无偏性**
*   **(1) 求矩估计**：硬算积分求期望。$E(X) = \int_0^\theta x \frac{1}{2\theta} dx + \int_\theta^1 x \frac{1}{2(1-\theta)} dx = \frac{\theta}{4} + \frac{1+\theta}{4} = \frac{2\theta+1}{4}$。
令 $\bar{X} = \frac{2\theta+1}{4}$，解得 $\hat{\theta} = \frac{4\bar{X}-1}{2}$。红笔完全正确！
*   **(2) 判断 $4\bar{X}^2$ 的无偏性**：
$E(4\bar{X}^2) = 4 \cdot E(\bar{X}^2) = 4 [D(\bar{X}) + (E\bar{X})^2] = 4 [\frac{D(X)}{n} + \mu^2]$。
因为 $\frac{D(X)}{n} > 0$，所以 $4[\frac{D(X)}{n} + \mu^2]$ 显然不可能恰好等于 $\theta^2$。红笔逻辑清晰。

**9. 神仙组合 $T = \bar{X}^2 - \frac{1}{n}S^2$ (考察卡方分布方差)**
*   **(1) 证明无偏**：$E(T) = E(\bar{X}^2) - \frac{1}{n}E(S^2) = [D(\bar{X}) + (E\bar{X})^2] - \frac{1}{n}\sigma^2 = [\frac{\sigma^2}{n} + \mu^2] - \frac{\sigma^2}{n} = \mu^2$。得证！
*   **(2) 求 $D(T)$**：这是全场最佳的一道题，你的红笔写得堪称标答！
核心思路是利用正态总体下 $\bar{X}$ 与 $S^2$ 相互独立，所以 $D(T) = D(\bar{X}^2) + \frac{1}{n^2}D(S^2)$。
然后利用卡方分布的方差是自由度的两倍（$2n$）：
    *   $\sqrt{n}\bar{X} \sim N(0,1) \implies n\bar{X}^2 \sim \chi^2(1) \implies D(n\bar{X}^2) = 2 \implies D(\bar{X}^2) = \frac{2}{n^2}$。
    *   $\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$，代入 $\sigma=1$ 得 $(n-1)S^2 \sim \chi^2(n-1) \implies D((n-1)S^2) = 2(n-1) \implies D(S^2) = \frac{2}{n-1}$。
    最后相加：$\frac{2}{n^2} + \frac{1}{n^2} \cdot \frac{2}{n-1} = \frac{2}{n(n-1)}$。

**10. 构造差分的无偏估计**
*   **解析**：题目要让 $E(c \sum(X_{i+1} - X_i)^2) = \sigma^2$。
我们单独看一项：$E[(X_{i+1} - X_i)^2] = D(X_{i+1} - X_i) + [E(X_{i+1} - X_i)]^2$。
因为 $X$ 是独立的简单随机样本：
1) 期望：$E(X_{i+1} - X_i) = \mu - \mu = 0$
2) 方差：$D(X_{i+1} - X_i) = D(X_{i+1}) + D(-X_i) = \sigma^2 + (-1)^2\sigma^2 = 2\sigma^2$。（注意独立变量相减，方差是**相加**的！）
所以 $E[(X_{i+1} - X_i)^2] = 2\sigma^2 + 0 = 2\sigma^2$。
求和号里面有 $n-1$ 项，所以总的期望是 $c \cdot (n-1) \cdot 2\sigma^2$。
要让它等于 $\sigma^2$，解方程 $2c(n-1) = 1$，得到 $c = \frac{1}{2(n-1)}$。红笔完全正确。
