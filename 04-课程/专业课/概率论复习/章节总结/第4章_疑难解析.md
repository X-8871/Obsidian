# 第四章 难题解析与核心概念总结

这份文档为你解答了第41页的20、25、26题，以及第42页的第29题第(2)问。更重要的是，在解答25和26题之前，我为你详细梳理了概率论中最容易混淆的几个概念：**协方差、相关系数、独立、同分布**之间的爱恨情仇。

---

## 💡 核心知识点：Cov, $\rho_{XY}$, 独立，不相关的关系网

这几个概念是考试中最喜欢出选择题和判断题的地方。请务必牢记以下关系：

### 1. 基础定义
*   **协方差 $Cov(X,Y)$**：衡量两个变量变化趋势的同向程度。
    公式：$Cov(X,Y) = E(XY) - E(X)E(Y)$
*   **相关系数 $\rho_{XY}$**：将协方差进行了“标准化”，把值限制在了 $[-1, 1]$ 之间，用来衡量两个变量的**线性相关程度**。
    公式：$\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{D(X)D(Y)}}$
*   **同分布**：X 和 Y 拥有完全一样的概率分布（即 $F_X(x) = F_Y(y)$ 或 $f_X(x) = f_Y(y)$）。但这**不代表**它们之间有任何关系，更**不代表**它们的值相等。同分布最直接的结论是：它们的**期望、方差完全相等**（$E(X)=E(Y), D(X)=D(Y)$）。

### 2. “不相关”与 $Cov(X,Y)$、$\rho_{XY}$ 的等价关系
说 X 和 Y **不相关**（即没有线性关系），在数学上以下四个表述是**完全等价**的：
1.  $X$ 与 $Y$ 不相关
2.  $\rho_{XY} = 0$
3.  $Cov(X,Y) = 0$
4.  $E(XY) = E(X)E(Y)$
*(只要看到其中一个，就可以立刻推导出另外三个。)*

### 3. “独立”与“不相关”的单向箭头
这是重中之重，请记住这个口诀：**“独立一定不相关，不相关不一定独立”**。
*   **独立 $\Rightarrow$ 不相关**：如果 X 和 Y 互相独立，那么它们之间没有任何关系（当然也没有线性关系），所以 $Cov(X,Y)$ 必定为 0，相关系数 $\rho_{XY}$ 必定为 0。
*   **不相关 $\nRightarrow$ 独立**：不相关仅仅代表它们之间没有“直线”关系（线性关系），但它们可能存在非线性的曲线关系（比如 $Y = X^2$），此时它们依然是不独立的。
*   *(唯一特例：对于**二维正态分布**，独立和不相关是等价的！)*

## 📝 题目详细解析

### 【第41页】 题目 20
**【题目】** 将 $n$ 只球 ($1 \sim n$) 随机地放进 $n$ 个盒子 ($1 \sim n$) 中去，一只盒子装一只球。若一只球装入与球同号的盒子中，称为一个配对，记 $X$ 为总的配对数，求 $E(X)$。
**【解析】**
这道题用“指标变量法”最简单。
定义指示变量 $X_i$：
如果第 $i$ 号球正好放进了第 $i$ 号盒子里，则 $X_i = 1$；否则 $X_i = 0$。
那么总配对数就是所有 $X_i$ 的总和：$X = X_1 + X_2 + \dots + X_n$
对于任意一个球 $i$，它被放入第 $i$ 号盒子的概率是 $\frac{1}{n}$。
所以 $X_i$ 是一道 $0-1$ 分布题，其期望 $E(X_i) = 1 \times \frac{1}{n} + 0 \times (1 - \frac{1}{n}) = \frac{1}{n}$。
利用期望的线性性质（不需要管它们是否独立，期望都可以直接相加）：
$E(X) = E(X_1 + X_2 + \dots + X_n) = E(X_1) + E(X_2) + \dots + E(X_n) = n \times \frac{1}{n} = 1$。
**【答案】 $E(X) = 1$**

### 【第41页】 题目 25
**【题目】** 如果随机变量 X 和 Y 满足 $D(X-Y) = D(X+Y)$，则必有（ ）
(A) X 与 Y 独立   (B) X 与 Y 不相关   (C) $D(Y)=0$   (D) $D(X)D(Y)=0$
**【解析】**
利用方差的展开公式：
$D(X-Y) = D(X) + D(Y) - 2Cov(X,Y)$
$D(X+Y) = D(X) + D(Y) + 2Cov(X,Y)$
因为题目说它们相等，所以：
$D(X) + D(Y) - 2Cov(X,Y) = D(X) + D(Y) + 2Cov(X,Y)$
两边消去 $D(X)$ 和 $D(Y)$，得到 $-2Cov(X,Y) = 2Cov(X,Y)$，从而得出 **$Cov(X,Y) = 0$**。
根据我们刚才总结的知识点，$Cov(X,Y) = 0$ 就意味着 **X 与 Y 不相关**。注意，这里无法推出独立（不相关推不出独立）。
**【答案】 选 (B)**

### 【第41页】 题目 26
**【题目】** 设两个随机变量 X 和 Y 独立且**同分布**，若 $U=X-Y$, $V=X+Y$，则 U 与 V 之间必有（ ）
(A) 不独立   (B) 相关系数不为零   (C) 独立   (D) 相关系数为零
**【解析】**
问 U 和 V 的相关系数，我们就去求它们的协方差 $Cov(U,V)$。
利用协方差的分配律直接展开：
$Cov(U,V) = Cov(X-Y, X+Y)$
$= Cov(X,X) + Cov(X,Y) - Cov(Y,X) - Cov(Y,Y)$
因为 $Cov(X,X) = D(X)$，$Cov(Y,Y) = D(Y)$，并且 $Cov(X,Y) = Cov(Y,X)$，所以中间两项抵消：
$Cov(U,V) = D(X) - D(Y)$
又因为题目条件说 X 和 Y **同分布**，所以它们的方差必然相等：$D(X) = D(Y)$。
所以得出 $Cov(U,V) = 0$。
协方差为 0 等价于**相关系数为零**（不相关）。
**【答案】 选 (D)**

### 【第42页】 题目 29 第(2)问
**【题目】** 假设二维连续型随机变量 (X,Y) 在圆域 $x^2 + y^2 \le r^2$ 上服从联合均匀分布。求 X 与 Y 的相关系数 $\rho_{XY}$。
**【解析】**
求相关系数，首先要找协方差 $Cov(X,Y) = E(XY) - E(X)E(Y)$。
对于在原点对称的区域（比如圆）上的均匀分布，利用**对称性和奇偶性**可以秒杀期望的计算：
1.  **求 $E(X)$**：$E(X) = \iint_{D} x \cdot f(x,y) dx dy$。由于积分区域是一个圆，关于 $y$ 轴对称，而被积函数 $x$ 是关于 $x$ 的奇函数。所以积分必然为 0。即 $E(X) = 0$。
2.  **求 $E(Y)$**：同理，积分区域关于 $x$ 轴对称，被积函数 $y$ 是奇函数，所以 $E(Y) = 0$。
3.  **求 $E(XY)$**：$E(XY) = \iint_{D} xy \cdot f(x,y) dx dy$。被积函数 $xy$ 无论是对于 $x$ 还是对于 $y$ 都是奇函数，在对称域上的积分依旧是 0。即 $E(XY) = 0$。

带入协方差公式：
$Cov(X,Y) = E(XY) - E(X)E(Y) = 0 - 0 \times 0 = 0$。
既然协方差为 0，那么无论分母的标准差是多少（只要不是0），分子为0就决定了相关系数为 0。
**【答案】 $\rho_{XY} = 0$**

*(补充：在第(3)问中，虽然算出来它们不相关，但由于圆域的边界不是矩形边界，所以 $f(x,y) \neq f_X(x)f_Y(y)$，所以 X 和 Y 在圆域均匀分布上是**不独立**的。这也是“不相关推不出独立”的最经典的反例！)*
