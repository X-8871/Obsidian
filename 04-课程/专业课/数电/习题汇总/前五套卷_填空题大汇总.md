## 📝 试卷一（PDF 第 1 页，共 3 页）

**一、（20分）填空、选择题（每空1分）**

1. $(37)_{10} = (\underline{52})_7 = (\underline{00110111})_{8421BCD} = (\underline{110111})_{\text{格雷码}}$

2. 任意两个[[最小项]]的乘积恒等于 $\underline{0}$，全部[[最小项]]之和恒等于 $\underline{1}$

3. 若 $F(A,B,C) = A \oplus B \oplus C$，则 $F = \sum m(\underline{1,2,4,7})$；
   $\overline{F} = (\overline{A}+\overline{B})(B+\overline{C})(\overline{A}+C)$ 的[[最小项]]标准式为：$F = \sum m(\underline{0,2,3})$

4. 在数字电路中，通常利用场效应管的 $\underline{开关}$ 效应来实现逻辑门电路。

5. 比起结型场效应管的反偏 PN 结来说，绝缘栅场效应管的输入阻抗更 $\underline{大}$，功耗更 $\underline{低}$，集成度高，在大规模集成电路中应用更广泛。

6. 判断所示两个电路分别实现何种逻辑运算：左图为 $\underline{非}$ 运算，右图为 $\underline{与非}$ 运算。

7. 某一存储器的地址线为 $A_{14} \sim A_0$，数据线为 $D_3 \sim D_0$，其存储容量是 $\underline{128K}$ 位（即 $32K\times4$ 位，$2^{15}\times4 = 2^{17}$ 位）。

8. 用 256字×4位 RAM 芯片，扩展为 1024字×8位 RAM，则需要 $\underline{8}$ 片。

9. ROM 属于 $\underline{A}$（A. 组合逻辑电路　B. 时序逻辑电路）。

10. 术语 ASIC 表示 $\underline{C}$（A. 标准集成芯片应用　B. 应用系统集成电路　**C. 专用集成电路**）。

11. 有一个 ADC 电路，$u_{REF}=12V, n=4$，试分别求出：
    - 采用**舍尾**量化时的量化单位 $\Delta = \underline{3/4\,V}$（即 $12V/2^4 = 0.75V$）
    - 采用**四舍五入**量化方式时的量化单位 $\Delta = \underline{24/31\,V}$（即 $12V/(2^4-1) \approx 0.774V$）
    - 如果 $u_I = 6.28V$：舍尾量化输出编码为 $\underline{1000}$，四舍五入量化方式输出编码为 $\underline{1000}$

> [!TIP]
> **试卷一答案确认**（来自 PDF 第 5 页答案页）：  
> 题7答案：32K×4（或 128K，$2^{17}$）；题8答案：8；题9：A；题10：C；  
> 题11：0.75V，0.774V，1000，1000

## 📝 试卷二（PDF 第 9 页）

**一、填空和选择题（每空1分，共25分）**

1. 当逻辑函数有 n 个变量时，共有 $\underline{D}$ 个变量取值组合。
   （A. n　B. 2n　C. $n^2$　**D. $2^n$**）

2. $(0111\ 1000)_{8421BCD} = (\underline{1001110})_2 = (\underline{0100\ 1100})_{\text{格雷码}}$

3. 逻辑函数 $F = A \oplus (A \oplus B) = \underline{A}$（答案选 **A**，即 $F=B$？注意原卷选项 A 就是"B"，请以实际图为准）

4. 建立 ASM 图时，必须用 $\underline{D}$ 分开不能在同一个时钟周期内完成的寄存器操作。
   （A. 传输框　B. 判断框　C. 条件框　**D. 状态框**）

5. 当二输入与非门输入为 $\underline{A}$ 变化时，输出可能有竞争冒险。
   （**A. 01→10**　B. 00→10　C. 10→11　D. 11→01）

6. 已知函数 $F = (\overline{A \oplus D})\overline{B} + C$，则其反函数 $\overline{F} = \underline{(A \odot D + B)\overline{C}}$

7. 若某 ADC 取量化单位 $\Delta = V_{REF}/8$，规定 $0 \leq u_I < V_{REF}/8$ 时输出编码 000；则 $5V_{REF}/8 \leq u_I < 6V_{REF}/8$ 时，输出的二进制编码为 $\underline{B}$ = **101**

8. 某移位寄存器的时钟脉冲频率为 100KHz，欲将存放在该寄存器中的数**左移 8 位**，完成该操作需要 $\underline{B}$ 时间。
   （A. 10μs　**B. 80μs**　C. 100μs　D. 800ms）
   > **计算**：$T = 1/100KHz = 10\mu s$，左移 8 位需 8 个时钟周期，$8 \times 10\mu s = \mathbf{80\mu s}$✅（答案确认为 80μs，不是 90μs）

9. 将一个时间上连续变化的模拟量转换为时间上离散的模拟量的过程称 $\underline{A}$（**A. 采样**　B. 量化　C. 保持　D. 编码）

10. 要构成容量为 4K×8 的 RAM，需要 $\underline{D}$ 片容量为 256×4 的 RAM。
    （A. 2　B. 4　C. 8　**D. 32**）
    > **计算**：$(4096 \times 8)/(256 \times 4) = 32768/1024 = 32$ 片

11. 随机存取存储器 RAM 中的内容，当电源断掉后又接通，存储器中的内容 $\underline{C}$。
    （A. 全部改变　B. 全部为1　**C. 不确定**　D. 保持不变）

## 📝 试卷三（PDF 第 19 页）

**一、填空选择题（20分，每空1分）**

1. 按十进制数 0, 1, 2, 3 的顺序写出对应**两位循环码**的码型：$\underline{00}$、$\underline{01}$、$\underline{11}$、$\underline{10}$

   - 其反函数 $\overline{F} = \underline{\overline{ABC} + (\overline{A}+\overline{B}+\overline{C}) = 1}$
   - 其对偶函数 $F' = \underline{ABC + (\overline{A}+\overline{B}+\overline{C}) = 1}$

3. 逻辑函数化简的常用方法有 $\underline{公式法}$ 和 $\underline{[[卡诺图]]法}$

4. JK 触发器的特征方程为 $Q^{n+1} = \underline{(J\overline{Q^n}+\overline{K}Q^n)CP\downarrow}$

5. 由与非门构成的基本 RS 触发器的约束条件是 $\underline{S+R=1}$（不允许 $\overline{S}=\overline{R}=0$ 同时有效）

6. n 个输入变量，可以构成[[最小项]]的[[最小项]]编号最大为 $\underline{D}$（$2^n-1$）
   （A. 2n　B. $2^n$　C. n　**D. $2^n-1$**）

7. PAL 器件在结构上的特点是：与阵列**可编程**，或阵列**固定**

8. 将 1K×4ROM 扩展为 8K×8ROM，需用 1K×4ROM $\underline{C}$ 片。
   （A. 4片　B. 8片　**C. 16片**　D. 32片）
   > **计算**：字扩展（$8K/1K=8$）× 位扩展（$8/4=2$）= **16 片**

9. 对 MSI 计数器，若 $\overline{CR}=0$，无论 CP 信号处于何状态，计数器立即清零，该清零方式称为**异步清零**；MSI 计数器 [[74161]] 的清零方式为**异步清零**

10. 在 A/D 转换器中，已知 Δ 是量化单位，若采用"舍尾"方法划分量化电平，则最大量化误差为 $\underline{C}$（$1\Delta$）
    （A. 1/4　B. 1/2　**C. 1**　D. 2）

11. 一个 8 位 D/A 转换器，当输入为 10000001 时输出电压为 5V，则输入为 01010000 时，输出电压为 $\underline{0.505}$ V
    > **计算**：10000001 = 129（十进制），对应 5V，则每单位 = 5V/129。01010000 = 80，输出 = 80×5V/129 ≈ 3.1V（注：此处答案以原卷为准）

12. 数字系统中，可以用**状态转移表**或**状态转移图**来描述控制器。
