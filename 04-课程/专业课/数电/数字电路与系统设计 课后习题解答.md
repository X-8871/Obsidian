## 3.7 (3) 组合逻辑电路设计 (最小与非门数)

**题目要求**：在双轨输入条件下，用最少与非门设计函数 $F(A,B,C,D) = \sum m(2,5,6,8,9,12) + \sum \phi(0,4,7,10)$。

### 解答步骤：
1. **卡诺图化简**：
  将最小项 $m$ 和任意项 $\phi$ 填入卡诺图，经过圈组化简得到最简与或表达式：
  - 包含 $m_5, m_6$ 的圈（结合 $\phi_4, \phi_7$）：$\bar{A}B$
  - 包含 $m_8, m_{12}$ 的圈（结合 $\phi_0, \phi_4$）：$\bar{C}\bar{D}$
  - 包含 $m_2$ 的圈（结合 $\phi_0, m_8, \phi_{10}$）：$\bar{B}\bar{D}$
  - 包含 $m_9$ 的圈（结合 $m_8$）：$A\bar{B}\bar{C}$

  得到最简与或式：
  $$F = \bar{A}B + \bar{C}\bar{D} + \bar{B}\bar{D} + A\bar{B}\bar{C}$$

2. **与非门转换**：
  利用摩根定律，将与或式转换为与非-与非形式：
  $$F = \overline{ \overline{\bar{A}B} \cdot \overline{\bar{C}\bar{D}} \cdot \overline{\bar{B}\bar{D}} \cdot \overline{A\bar{B}\bar{C}} }$$

3. **结论**：
  由于是双轨输入（$A, \bar{A}, \dots$ 均可直接使用），共需 **4个** 与非门作为第一级（实现各项），以及 **1个** 4输入与非门作为第二级（实现求和），总计 **5个与非门**。

## 3.9 血型匹配逻辑电路设计

**题目要求**：根据血型匹配关系图（O型可输给所有血型；A型可输给A、AB；B型可输给B、AB；AB型只能输给AB），设计一个逻辑电路判断输血是否符合规定。

### 解答步骤：
1. **状态编码**：
  设供血者为 $X$ ($x_1, x_0$)，受血者为 $Y$ ($y_1, y_0$)。采用抗原分布编码（最简设计）：
  - **O型**：00（无 A、B 抗原）
  - **A型**：01（含 A 抗原）
  - **B型**：10（含 B 抗原）
  - **AB型**：11（含 A、B 抗原）

2. **匹配规则**：
  供血者能输血给受血者的条件是：受血者必须拥有供血者所含的所有抗原。
  即：$x_1 \le y_1$ 且 $x_0 \le y_0$。

3. **逻辑表达式**：
  判断结果 $F = 1$ 表示匹配：
  $$F = (\bar{x_1} + y_1)(\bar{x_0} + y_0)$$
  展开为与或式：
  $$F = \bar{x_1}\bar{x_0} + \bar{x_1}y_0 + \bar{x_0}y_1 + y_1y_0$$

4. **电路实现**：
  该电路可用 2 个或门和 1 个与门实现，或使用标准与非门逻辑实现。

## 3.10 电话编码控制电路设计

**题目要求**：对火警（Fire）、急救（Emergency）、普通（General）三种电话进行编码，优先级依次降低。编码分别为 11、10、01。

### 解答步骤：
1. **真值表设计**（设输入为 $I_F, I_E, I_G$，输出为 $Y_1, Y_0$）：
  | $I_F$ | $I_E$ | $I_G$ | $Y_1$ | $Y_0$ | 备注 |
  | :---: | :---: | :---: | :---: | :---: | :--- |
  | 1 | X | X | 1 | 1 | 火警最高优先 |
  | 0 | 1 | X | 1 | 0 | 急救次之 |
  | 0 | 0 | 1 | 0 | 1 | 普通最低 |
  | 0 | 0 | 0 | 0 | 0 | 无呼叫 |

2. **逻辑表达式**：
  - $Y_1 = I_F + \bar{I_F}I_E = I_F + I_E$
  - $Y_0 = I_F + \bar{I_F} \bar{I_E} I_G$

## 3.13 使用 74151 (8选1选器) 实现函数

**方法说明**：将 $A, B, C$ 连接至选择端 $S_2, S_1, S_0$，输入端 $D_i$ 根据 $D$ 的取值确定。

### (1) $F(A,B,C,D) = \sum m(1,2,4,7)$
- $D_0 = D$ （对应 $m_0, m_1$ 中的 $m_1$）
- $D_1 = \bar{D}$ （对应 $m_2, m_3$ 中的 $m_2$）
- $D_2 = \bar{D}$ （对应 $m_4, m_5$ 中的 $m_4$）
- $D_3 = D$ （对应 $m_6, m_7$ 中的 $m_7$）
- $D_4 \sim D_7 = 0$

### (3) $F(A,B,C,D) = \bar{A}BC + \bar{B}CD + A\bar{C}\bar{D}$
- 展开为最小项：$F = \sum m(3,6,7,8,11,12)$
- $D_0 = 0, D_1 = D (m_3), D_2 = 0, D_3 = 1 (m_6, m_7)$
- $D_4 = \bar{D} (m_8), D_5 = D (m_{11}), D_6 = \bar{D} (m_{12}), D_7 = 0$

### (4) $F(A,B,C,D) = \sum m(0,3,12,13,14) + \sum \phi(7,8)$
- $D_0 = \bar{D} (m_0), D_1 = D (m_3), D_2 = 0, D_3 = D$ (利用 $\phi_7$)
- $D_4 = 0$ (利用 $\phi_8$), $D_5 = 0, D_6 = 1 (m_{12}, m_{13}), D_7 = \bar{D} (m_{14})$

## 3.14 (1) 使用 1/2 74153 (4选1选器) 实现函数

**题目要求**：$F(A,B,C,D) = \sum m(1,2,4,7,15)$。
**方法说明**：选 $A, B$ 为选择端 $S_1, S_0$，输入 $D_i$ 为 $C, D$ 的函数。
- $D_0 (AB=00)$：包含 $m_1, m_2 \implies \bar{C}D + C\bar{D} = C \oplus D$
- $D_1 (AB=01)$：包含 $m_4, m_7 \implies \bar{C}\bar{D} + CD = \overline{C \oplus D}$
- $D_2 (AB=10)$：无最小项 $\implies 0$
- $D_3 (AB=11)$：包含 $m_{15} \implies CD$

## 3.16 使用 74283 实现 8421 码求和电路

**设计思路**：
1. **首级相加**：使用一个 74283 对两个 8421 码 $A, B$ 进行二进制相加。
2. **进位修正**：如果二进位和 $S > 9$ 或产生二进制进位 $C_4=1$，则需修正。
  - 修正条件：$L = C_4 + S_3S_2 + S_3S_1$。
3. **加6修正**：当 $L=1$ 时，产生十进制进位，并向二进制和数加上 **0110 (6)**。
  - 可用逻辑门或第二个 74283 实现：将 $L$ 接至第二个加法器的 $B_2, B_1$，从而实现加 6。

## 3.19 使用 7485 和 74157 实现数值比较并输出大数

**设计思路**：
1. **比较部分**：将 4位二进制数 $A, B$ 送入 **7485 数值比较器**，获取 $B > A$ 的输出信号。
2. **选择部分**：使用 **74157 四2选1数据选择器**。
  - 将 $A$ 接至 $I_0$ 端口，将 $B$ 接至 $I_1$ 端口。
  - 将 7485 的 $B > A$ 输出信号接至 74157 的 **选择端 S**。
3. **结果**：
  - 若 $B > A$，则 $S=1$，选择器输出 $B$。
  - 若 $B \le A$，则 $S=0$，选择器输出 $A$。
  - 最终输出即为 $A, B$ 中的较大值。
