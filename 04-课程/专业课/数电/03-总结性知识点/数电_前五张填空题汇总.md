# 数电期末冲刺：前五套卷 填空题汇总

> 提取自《数电A-多试题.pdf》前五套试卷的填空题部分。

---

## 试卷一

1. **进制转换与编码**：
   $(37)_{10} = (52)_7 = (00110111)_{8421BCD} = (110111)_{\text{格雷码}}$。

2. **逻辑代数基础**：
   任意两个最小项的乘积恒等于 **0**，全部最小项之和恒等于 **1**。

3. **最小项推导**：
   若 $F(A,B,C) = A \oplus B \oplus C$，则 $F = \sum m(1, 2, 4, 7)$；
   $F = (\overline{A}+\overline{B})(B+\overline{C})(\overline{A}+C)$ 的最小项标准式为：$F = \sum m(0, 2, 3)$。

4. **门电路物理基础**：
   在数字电路中，通常利用场效应管的 **开关** 效应来实现逻辑门电路。

5. **场效应管特性对比**：
   比起结型场效应管的反偏 PN 结来说，绝缘栅场效应管的输入阻抗更 **大**，功耗更 **低**，集成度高，在大规模集成电路中应用更广泛。

6. **CMOS 电路判断**：
   （根据图示推断）左图为 **非（NOT）** 运算，右图为 **与非（NAND）** 运算。

7. **存储器容量计算**：
   某一存储器的地址线为 $A_{14} \sim A_0$，数据线为 $D_3 \sim D_0$，其存储容量是 **128K** 位。

8. **存储器扩展**：
   用 256字×4位 RAM 芯片，扩展为 1024字×8位 RAM，则需要 **8** 片。

9. **器件分类**：
   ROM 属于 **组合逻辑电路**（在没加时序宏单元的情况下其本质是组合逻辑）。

10. **术语解析**：
    术语 ASIC 表示 **专用集成电路 (Application Specific Integrated Circuit)**。

11. **ADC 量化单位计算**：
    有一个 ADC 电路，$U_{REF}=12V, n=4$。
    - 采用舍尾量化时的量化单位 $\Delta = \frac{3}{4}V$（即 $12/2^4$）；
    - 采用四舍五入量化方式时的量化单位 $\Delta = \frac{24}{31}V$；
    - 如果 $u_I = 6.28V$，舍尾量化输出编码为 **1000**，四舍五入量化方式输出编码为 **1000**。

## 试卷二

1. **组合数量**：
   当逻辑函数有 n 个变量时，共有 **$2^n$** 个变量取值组合。

2. **进制与编码转换**：
   $(0111\ 1000)_{8421BCD} = (1001110)_2 = (0100\ 1100)_{\text{格雷码}}$。

3. **异或化简**：
   逻辑函数 $F = A \oplus (A \oplus B) = B$。

4. **ASM 图绘制规范**：
   建立 ASM 图时，必须用 **状态框** 分开不能在同一个时钟周期内完成的寄存器操作。

5. **竞争冒险**：
   当二输入与非门输入为 **01变10** 变化时，输出可能有竞争冒险。

6. **反函数推导（摩根定律）**：
   已知函数 $F = (\overline{A \oplus D})\overline{B} + C$，则其反函数 $\overline{F} = ((A \odot D) + B)\overline{C}$。

7. **ADC 编码边界**：
   若某 ADC 取量化单位 $\Delta = V_{REF}/8$，输入电压 $u_I$ 在 $0 \leq u_I < V_{REF}/8$ 时编码为 000；则在 $5V_{REF}/8 \leq u_I < 6V_{REF}/8$ 时，输出的二进制编码为 **101**（对应十进制的 5）。

8. **移位寄存器耗时计算**：
   某移位寄存器的时钟脉冲频率为 100KHz，欲将存放在该寄存器中的数左移 8 位，完成该操作需要 **$80 \mu s$** 时间。（$T = 1/f = 10\mu s$，8 位即 $8 \times 10 = 80\mu s$）。

9. **模数转换步骤**：
   将一个时间上连续变化的模拟量转换为时间上离散的模拟量的过程称之为 **采样**。

10. **存储器片数计算**：
    要构成容量为 4K×8 的 RAM，需要 **32** 片容量为 256×4 的 RAM。（$(4096 \times 8) / (256 \times 4) = 16 \times 2 = 32$）。

11. **RAM 掉电特性**：
    随机存取存储器 RAM 中的内容，当电源断掉后又接通，存储器中的内容 **不确定（丢失）**。

## 试卷三

1. **循环码序列**：
   按十进制数 0, 1, 2, 3 的顺序，写出对应两位循环码的码型：**00、01、11、10**。

2. **反函数与对偶式求法**：
   逻辑函数 $F = (A+B+C)\overline{A B C} = 0$
   其反函数 $\overline{F} = ABC + (\overline{A}+\overline{B}+\overline{C}) = 1$。
   其对偶函数 $F' = ABC + (\overline{A}+\overline{B}+\overline{C}) = 1$。

3. **化简方法**：
   逻辑函数化简的常用方法有 **公式法** 和 **卡诺图法**。

4. **触发器特征方程**：
   JK 触发器的特征方程为 $Q^{n+1} = (J\overline{Q^n} + \overline{K}Q^n)CP\downarrow$。

5. **约束条件**：
   由与非门构成的基本 RS 触发器的约束条件是 $S+R=1$（即不允许同时给 0）。

6. **最小项编号范围**：
   n 个输入变量，可以构成最小项的最小项编号最大为 **$2^n - 1$**。

## 试卷四

> 注：本卷填空部分多以选择填空形式出现。

1. **逻辑代数法则**：
   以下表达式中符合逻辑运算法则的是 **A+1=1**。

2. **带小数的进制转换**：
   $(11101.1)_2 = (29.5)_{10} = (0010\ 1001.0101)_{8421BCD}$。

3. **JK 触发器特性**：
   JK 触发器的激励取 **J=Q, K=Q** 组合时，触发器不按 $Q^{n+1}=Q^n$ 工作。

4. **竞争冒险消除**：
   函数 $F = \overline{A}C + AB + B\overline{C}$，当变量取值为 **A=0, B=0, C=0** 时，将不会出现冒险现象。

5. **译码器逻辑实现**：
   用 3-8 线译码器 74LS138 和辅助逻辑门实现逻辑函数 $Y = A_2 + \overline{A_2}\overline{A_1}$，可选用 **与非门**。

## 试卷五

1. **消除静态冒险**：
   逻辑表达式 $F = A\overline{B} + \overline{A}C + B\overline{C}$ 中，变量 A、B、C 均具有竞争能力。不存在冒险的变量组合为 **B**。

2. **小数保留精度转换**：
   转换下列数，要求转换后保持原精度：$(5.25)_{10} = (101.0100000)_2$。

3. **PLD 器件对比**：
   PAL 和 GAL 的相同之处是基本结构都是 **与** 阵列可编程；不同之处是 **GAL** 的输出结构可由用户编程确定。

4. **D/A 转换电压计算**：
   有一个 4 位的 D/A 转换器，设它的满刻度输出电压为 10V（对应 1111）。当输入数字量为 1101（十进制 13）时，输出电压为 **8.125V** 左右。（$10V \times \frac{13}{16} = 8.125V$）。

5. **ROM 的扩展计算**：
   把 1K×4 容量的 RAM 2114 扩展为 16K×16 的 RAM，则需 **64** 片 RAM2114，和 **4/16** 线译码器。（$(16 \times 16) / (1 \times 4) = 64$）。

6. **逻辑器件选型**：
   实现单输入、多输出逻辑函数，应选：**数据分配器**。
