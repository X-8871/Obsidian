# 第一个深度学习工程总结：MNIST 手写数字识别

## 项目概述
本项目是从零入门深度学习的第一个完整实战工程。通过摒弃“调包侠”的黑盒式学习方法，以**数学公式推导**与 **C 语言底层映射**为核心视角，完整经历了从神经元基础定义、模型训练、病理诊断，直至工业级 Web 部署的全生命周期。

## 核心里程碑回顾

### 阶段一：打破黑盒，理解本质
*   通过手推 $Z = WX+b$ 和前向/反向传播公式，明确了神经网络在底层不过是“巨型多维浮点数数组”之间的批量乘加运算。
*   将 PyTorch 中的 `nn.Module` 和 `nn.Linear` 成功解构为 C 语言中的“指针结构体”和动态内存分配（malloc）。

### 阶段二：基座搭建与初步训练
*   成功编写了 `NeuralNet` 结构体，构建了一层拥有 128 个神经元的隐藏层和基础的 SGD 优化器。
*   利用 `DataLoader` 构建了批量数据流。
*   通过初次训练，模型在基础配置下成功跑通，并在测试集取得了及格线（约 90%）的初步结果。

### 阶段三：科学诊断与终极优化 (学霸之路)
*   **引入验证集 (Validation Set)**：抛弃盲目训练，构建了严谨的模拟考评价体系，实现了“训练集 / 验证集 / 测试集”的三级分离。
*   **临床切除高偏差**：通过扩充网络深度（双隐藏层 256 -> 128），解决了模型脑容量不足的欠拟合问题。
*   **精准打击高方差**：引入了反死记硬背的神器 `nn.Dropout(0.2)`，在训练中随机屏蔽 20% 神经元，彻底锁死过拟合。
*   **更换高级引擎**：将 SGD 更换为带有动量和自适应步长的 `Adam` 优化器，在极少的 5 次迭代循环内，将测试成绩飙升至 **96.87%**。

### 阶段四：跨越鸿沟的工业级部署
*   **模型持久化**：使用 `torch.save` 将矩阵权重的训练成果提取并存为独立的 `.pth` 物理文件，打通了离线加载推理。
*   **引入计算机视觉清洗**：在真实照片识别翻车后，深刻理解了数据分布差异（Domain Shift）的灾难性后果。随后引入了 OpenCV C++ 引擎，搭建了包含 OTSU 二值化、轮廓检测、居中与缩放的工业级视觉清洗管道。
*   **打通 Web 前后端交互**：运用 Client-Server 架构，通过 Flask 搭建后端死循环守护进程（霸占 5000 端口），与前端极客风 HTML 网页实现完美协同。完成从“实验脚本”向“产品级软件”的飞跃。

## 学习感悟与心得
真正的深度学习不在于调取了什么神仙框架，而在于理解数据的流动、显存的管理、导数的链式法则以及对非结构化数据的敬畏心。这个项目不仅是一个图像识别 Demo，更是我们用底层程序员的严谨视角，重塑整个 AI 世界观的第一块坚实地基。
