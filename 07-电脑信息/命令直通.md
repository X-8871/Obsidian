# 命令直通

> **用途**：在该 Vault 环境中经验证可稳定运行的读写命令合集。  
> **环境**：Git Bash + Python 3.12, 路径含中文/空格, VM 路径映射  
> **创建**：2026-06-03, 首次维护

---

## 一、核心原则

| 经验 | 说明 |
|------|------|
| **优先 Python** | Git Bash 管道+`while read`+中文文件名极易触发 `syntax error` 或 `Stream closed`，逻辑复杂时写 `.py` 脚本最稳 |
| **命令要短** | 单次命令输出过长会导致工具断开（`Stream closed`），核心数据分段获取 |
| **路径引号** | 中文和空格路径必须用双引号包裹 |
| **`python` 不是 `python3`** | 系统只有 `python` 命令 |
| **VM 路径映射** | 在 Bash 中 `C:\...` 映射为 `/sessions/serene-practical-pasteur/mnt/...` |

## 二、Vault 文件读写直通命令

### 2.1 统计笔记总数与目录分布

```bash
# 可靠：用 find + wc
find "." -name "*.md" -not -path "./.obsidian/*" -not -path "./.claude/*" | wc -l

# 分目录统计（替换目录名）
for d in 00-索引 01-Daily/Daily 01-Daily/Weekly 02-项目 03-知识库; do
  count=$(find "$d" -name "*.md" -not -path "./.obsidian/*" 2>/dev/null | wc -l)
  echo "$d: $count"
done
```

### 2.2 遍历所有笔记列表

```bash
# 列出所有笔记（排除系统目录）
find "." -name "*.md" -not -path "./.obsidian/*" -not -path "./.claude/*" -not -name "CLAUDE.md" | sort
```

### 2.3 验证所有 [[wikilink]] 有效性

**推荐方法（Python，最稳定）：**

```bash
# 写到临时脚本再执行，避免 heredoc 中文编码问题
python -c "
import os, glob, re

all_files = []
for f in glob.glob('**/*.md', recursive=True):
    if '.obsidian' in f or '.claude' in f or f == 'CLAUDE.md':
        continue
    all_files.append(f)

file_content = {}
for f in all_files:
    try:
        with open(f, 'r', encoding='utf-8') as fh:
            file_content[f] = fh.read()
    except:
        pass

basename_map = {}
for f in all_files:
    base = os.path.splitext(os.path.basename(f))[0]
    basename_map[base] = f

broken = []
for f, content in file_content.items():
    links = re.findall(r'\[\[([^\]]+)\]\]', content)
    for link in links:
        clean = link.split('|')[0]
        # Try basename match
        target = basename_map.get(clean)
        # Try relative path (from current file or vault root)
        if not target:
            rel = os.path.normpath(os.path.join(os.path.dirname(f), clean + '.md'))
            if os.path.exists(rel):
                target = rel
        if not target:
            root = clean + '.md'
            if os.path.exists(root):
                target = root
        if not target:
            broken.append((f, clean))

if broken:
    for f, link in broken:
        print(f'BROKEN: {f} -> [[{link}]]')
else:
    print('All links valid ✅')
"
```

### 2.4 找孤岛笔记（无入链）

```bash
python -c "
import os, glob, re

all_files = []
for f in glob.glob('**/*.md', recursive=True):
    if '.obsidian' in f or '.claude' in f or f == 'CLAUDE.md':
        continue
    all_files.append(f)

file_content = {}
for f in all_files:
    try:
        with open(f, 'r', encoding='utf-8') as fh:
            file_content[f] = fh.read()
    except:
        pass

basename_map = {}
for f in all_files:
    base = os.path.splitext(os.path.basename(f))[0]
    basename_map[base] = f

orphans = []
for base, path in sorted(basename_map.items()):
    ref_count = 0
    for other_f, content in file_content.items():
        if other_f == path:
            continue
        if re.search(r'\[' + re.escape(base) + r'[\]\|]', content):
            ref_count += 1
    if ref_count == 0:
        orphans.append(path)

for p in orphans:
    if 'Temptlaters/' not in p and '99-模板/' not in p:
        print(f'ORPHAN: {p}')
"
```

### 2.5 索引一致性检查（索引 vs 实际文件）

```bash
python -c "
import os, glob, re

idx_dir = '00-索引'
for idx_file in glob.glob(idx_dir + '/*.md'):
    if '总索引' in idx_file or '类型规范' in idx_file or '结构说明' in idx_file:
        continue
    content = open(idx_file, encoding='utf-8').read()
    links = re.findall(r'\[\[([^\]]+)\]\]', content)
    links = [l.split('|')[0] for l in links]
    # Map out the domain from filename
    print(f'--- {idx_file} ({len(links)} links) ---')
"
```

## 四、常见问题与规避

| 问题 | 表现 | 规避 |
|------|------|------|
| `while read` + 管道 + 中文 | syntax error / 变量丢失 | 写 Python 脚本 |
| 输出太长 | Stream closed | 分段获取或用 Python 精确控制 |
| heredoc 中文乱码 | Git Bash 传参问题 | 写 `.py` 文件到 outputs 再执行 |
| `grep -oP` 不支持 | Git Bash grep 差异 | 改用 `grep -o` 或 Python `re.findall` |
| 带空格的路径 | 命令找不到文件 | 始终用双引号包裹 |
