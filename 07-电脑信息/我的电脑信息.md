## 编码与字符限制

- **系统编码**: 中文(中国), `zh-cn`, UTF-8
- **Git Bash 编码问题**: heredoc 传中文会乱码，优先写 `.py` 文件再执行
- **路径有中文和空格**: Bash 命令中必须用引号包裹，如 `"C:\Users\22061\Documents\Obsidian Vault\某文件.md"`
- **Python print 中文**: 有时会被 WSL/Bash 层截断，避免在 Python 脚本中 print 中文文件名
- **wmic 输出乱码**: WMIC 命令在 Git Bash 中输出为 UTF-16 LE 编码，需用 `cmd.exe /c "wmic ..."` 或 PowerShell 替代

## 已知问题

### 自动重启 / GPU TDR 蓝屏（2026-06-13，当前问题）
- **现象**: 电脑频繁"自动重启"，实际是 NVIDIA GPU TDR 超时导致蓝屏后自动重启
- **根因**: BugCheck `0x00000116`（`VIDEO_TDR_TIMEOUT_DETECTED`），`nvlddmkm` 报告 TDR on GPUID:100
- **触发时间**: 6/12 更换主板后开始，之前从未出现过此蓝屏码
- **已排除**: 外接显示器/HDMI、Windows Update、驱动版本（回退无效）
- **最可能原因**: 新主板上焊接的 GPU 有硬件缺陷（换主板 = 换 GPU）
- **已采取措施**: 关闭蓝屏后自动重启（`AutoReboot=0`），等待下次蓝屏拍照
- **待处理**: 下次蓝屏后决定是否 DDU 重装驱动或禁用 GPU 测试

### HP 黑屏/闪屏问题（2026-06-03 ~ 06-07，已通过更换主板解决）
- **现象**: 接外接显示器时系统误判合盖，屏幕熄灭/闪屏
